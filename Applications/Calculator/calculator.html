📦 Top element (auto height)

This block’s height is calculated automatically based on its content. Add or remove text — the height adapts without any JavaScript.

🧩 Minimal example: pure CSS + legacy -webkit-box
Works in Safari 5 / old iOS and other vintage WebKit browsers.

✅ Bottom element — stretches the rest of the height
Using -webkit-box-flex: 1 inside an old flexible box container.

Even if you resize the window, the bottom block dynamically fills whatever vertical space remains after the top block's content.

📐 How it works:
→ Parent .old-flex-container uses display: -webkit-box
→ -webkit-box-orient: vertical stacks children vertically.
→ Top child takes natural content height (box-flex: 0 by default).
→ Bottom child uses -webkit-box-flex: 1 to consume leftover height.

★ no JavaScript required — pure CSS, works in Safari 5 (and other legacy WebKit browsers).

↕️ resize window → bottom stretches, top keeps auto height
